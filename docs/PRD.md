# 產品需求文檔 (PRD)

> 版本：2.0.0 | 更新：2026-01-25

## 產品概述

**TermLauncher** 是一個 Windows 桌面應用，讓使用者能夠快速開啟 Windows Terminal 並自動進入指定的工作目錄。

### 目標用戶

- 開發者（頻繁在不同專案目錄間切換）
- DevOps 工程師
- 使用 WSL 進行開發的 Windows 用戶

### 解決的痛點

| 痛點                       | 解決方案                     |
| -------------------------- | ---------------------------- |
| 每次開終端都要手動 cd      | 一鍵開啟並進入指定目錄       |
| 專案目錄散落各處           | 集中管理、分組分類           |
| WSL 路徑轉換麻煩           | 自動轉換 Windows → WSL 路徑  |
| 頻繁使用的目錄難以快速存取 | 最近使用 + 最愛 + 全域快捷鍵 |

## 功能概覽

### 核心功能

- **目錄管理**：新增、編輯、刪除目錄（支援自訂 Emoji 圖示）
- **終端啟動**：WSL Ubuntu、PowerShell、Git Bash、自訂終端
- **路徑轉換**：Windows 路徑自動轉為 WSL 格式

### 組織功能

- **群組分類**：建立群組，目錄按群組顯示
- **最愛標記**：常用目錄加入最愛，獨立 Tab 快速存取
- **最近使用**：記錄開啟歷史，可設定保留數量上限

### 便利功能

- **搜尋過濾**：依名稱、路徑、群組搜尋
- **拖拉排序**：最愛、群組、目錄皆支援
- **全域快捷鍵**：Alt+Space 顯示/隱藏視窗（可自訂）
- **系統托盤**：最小化到托盤，隨時待命

### 個人化

- **主題切換**：深色/淺色主題
- **多語系**：繁體中文、English
- **匯出匯入**：配置檔備份與還原

## 介面結構

5 個 Tab 頁面：

1. **🕐 最近** - 最近開啟的目錄
2. **⭐ 最愛** - 標記為最愛的目錄
3. **🏷️ 群組** - 群組管理
4. **📁 目錄** - 所有目錄（按群組分類）
5. **⚙️ 設定** - 應用設定、終端管理、資料管理

## 非功能需求

| 項目     | 需求                            |
| -------- | ------------------------------- |
| 啟動時間 | < 2 秒                          |
| 記憶體   | < 150 MB                        |
| 作業系統 | Windows 10 (1903+) / Windows 11 |
| 相依     | Windows Terminal                |
