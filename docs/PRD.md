## 1. 產品概述

### 1.1 產品簡介

TermLauncher 是一個 Windows 桌面應用程式，讓使用者能夠快速開啟 Windows Terminal 並自動進入指定的工作目錄。支援 WSL Ubuntu 和 PowerShell 兩種終端環境。

### 1.2 目標用戶

- 開發者（需要頻繁在不同專案目錄間切換）
- DevOps 工程師
- 使用 WSL 進行開發的 Windows 用戶

### 1.3 解決的問題

| 痛點                           | 解決方案                         |
| ------------------------------ | -------------------------------- |
| 每次開啟終端都要手動 cd 到目錄 | 一鍵開啟並進入指定目錄           |
| 專案目錄散落各處難以管理       | 集中管理、分組分類               |
| WSL 路徑轉換麻煩               | 自動將 Windows 路徑轉為 WSL 路徑 |
| 頻繁使用的目錄難以快速存取     | 最近使用記錄 + 全域快捷鍵        |

## 3. 非功能需求

### 3.1 效能需求

| 項目       | 指標     |
| ---------- | -------- |
| 啟動時間   | < 2 秒   |
| 記憶體占用 | < 150 MB |
| 配置讀取   | < 100 ms |

### 3.2 相容性需求

| 項目     | 需求                            |
| -------- | ------------------------------- |
| 作業系統 | Windows 10 (1903+) / Windows 11 |
| 終端程式 | Windows Terminal                |
| WSL 版本 | WSL 2                           |
| Node.js  | v16+ (開發環境)                 |

### 3.3 安全需求

- 配置檔僅儲存於本機，不上傳任何資料
- 不儲存任何敏感資訊（密碼、金鑰等）
