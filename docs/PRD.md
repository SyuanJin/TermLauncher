# 產品需求文檔 (PRD)

> 版本：2.3.0 | 更新：2026-02-11

## 產品概述

**TermLauncher** 是一個跨平台桌面應用，支援以任意應用程式開啟專案目錄 — 終端、編輯器、IDE、檔案總管等。Windows 完整支援，macOS/Linux 實驗性支援。

### 目標用戶

- 開發者（頻繁在不同專案目錄間切換）
- DevOps 工程師
- 使用 WSL 進行開發的 Windows 用戶

### 解決的痛點

| 痛點                       | 解決方案                     |
| -------------------------- | ---------------------------- |
| 每次開終端都要手動 cd      | 一鍵開啟並進入指定目錄       |
| 專案目錄散落各處           | 集中管理、分組分類           |
| WSL 路徑轉換麻煩           | 自動轉換 Windows → WSL 路徑  |
| 頻繁使用的目錄難以快速存取 | 最近使用 + 最愛 + 全域快捷鍵 |

## 功能概覽

### 核心功能

- **目錄管理**：新增、編輯、刪除目錄（支援自訂 Emoji 圖示）
- **啟動器支援**：WSL Ubuntu、PowerShell、Git Bash、檔案總管、自訂啟動器
- **路徑轉換**：Windows 路徑自動轉為 WSL 格式
- **啟動命令預覽**：啟動前預覽實際執行的命令與參數
- **路徑失效預警**：自動偵測並標示無效路徑，提供視覺指示

### 組織功能

- **群組分類**：建立群組，目錄按群組顯示
- **最愛標記**：常用目錄加入最愛，獨立 Tab 快速存取
- **最近使用**：記錄開啟歷史，可設定保留數量上限

### 便利功能

- **搜尋過濾**：依名稱、路徑、群組搜尋
- **拖拉排序**：最愛、群組、目錄皆支援
- **全域快捷鍵**：Alt+Space 顯示/隱藏視窗（可自訂）
- **系統托盤**：最小化到托盤，隨時待命
- **開機自動啟動**：支援 Portable 與安裝版（Portable 版本會提示路徑限制）
- **啟動時最小化**：啟動後自動最小化到系統托盤

### 個人化

- **主題切換**：深色/淺色主題
- **多語系**：繁體中文、English
- **Tab 文字顯示開關**：可切換 Tab 只顯示圖示或圖示加文字
- **進階匯出匯入**：選擇性匯出（啟動器、群組、目錄、最愛、設定）、合併/覆蓋匯入

### 可靠性

- **配置損壞偵測與備份**：自動偵測配置檔損壞，備份後重建預設配置
- **配置重設**：一鍵恢復所有設定為預設值
- **日誌清除**：手動清除日誌檔案

## 介面結構

6 個 Tab 頁面：

1. **🕐 最近** - 最近開啟的目錄
2. **⭐ 最愛** - 標記為最愛的目錄
3. **🏷️ 群組** - 群組管理
4. **📁 目錄** - 所有目錄（按群組分類）
5. **🚀 啟動器** - 啟動器偵測與管理
6. **⚙️ 設定** - 應用設定、資料管理

## 非功能需求

| 項目     | 需求                                                              |
| -------- | ----------------------------------------------------------------- |
| 啟動時間 | < 2 秒                                                            |
| 記憶體   | < 150 MB                                                          |
| 作業系統 | Windows 10 (1903+) / Windows 11（完整支援）                       |
| 跨平台   | macOS 10.15+、Linux（Ubuntu 20.04+ 等主流發行版）為實驗性支援     |
| 相依     | Windows Terminal（Windows）；macOS/Linux 使用系統內建或第三方終端 |
| 安全性   | Context Isolation、CSP、IPC 參數驗證、XSS 防護、路徑安全驗證      |
